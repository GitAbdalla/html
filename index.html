Introduction

Structurize.ai is a unified API to extract data from any file format. Instead of training custom models, you simply provide a JSON Schema, and our AI ensures the output matches it.

Authentication

All API requests require a Bearer Token. You can generate one in your Dashboard.

Authorization: Bearer sk_live_8392...

Endpoint

POST https://api.structurize.ai/v1/extract

Body Parameters:

  • file (string): Base64 encoded file.
  • fileType (string): 'image', 'excel', 'word'.
  • schema (object): The target JSON structure.
